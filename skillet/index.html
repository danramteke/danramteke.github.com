Skillet Timer

Shake it to start the timer.

When cooking steak, I set the timers for the cooking times that I want. When my skillet finishes pre-heating, I place it on the on the stove, place steaks on the skillet, and shake this app to start the timer. When the first timer finishes for the first side of steak, I flip the steaks, and shake the app to start the second timer. When the second timer is done, I put the skillet in the oven, and shake the app to start the third timer. After the steak finishes in the oven, I plate the steaks, wrapping them in aluminum foil to rest. And then I eat delicious steak way too fast.

My non-dominant hand can shake an iPhone. It's not so good at quickly pushing buttons. There's still a start button on this app. When all the timers have run their course, the Start button turns into a Reset button to reset the app. If you get stuck in the middle, you can reset all timers

To set a time on a timer, tap the timer while it isn't timing anything.
